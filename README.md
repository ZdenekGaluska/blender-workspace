# Blender Workspace

Personal Blender projects including procedural tools and original 3D artwork.
All models and renders are original work.

## Projects

### [Terrain Generator](terrain-generator/)
Blender addon that generates procedural terrain using fractal noise and 
configurable hill placement. Fully controllable from a sidebar UI panel.

### [Renders](renders/)
Collection of original 3D renders — environments, weapons, and architectural scenes.
